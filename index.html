Torque Wrench Calculator

Calculate torque with an extension and angle

Variable Description
Resulting Torque The torque with the extension (output/result)
Torque Value The initial torque without the extension
Total Length The combined length of the torque wrench and the extension
Original Length The length of the torque wrench without the extension
Extension Angle The angle of the extension in degrees
Note: Original Length = Torque Wrench Length (Center of Socket to Mid Handle)
Formula: Resulting Torque = Torque Value × (Total Length / Original Length) × cos(Extension Angle)
